# Blake Automator v4
> A fast, modern premium automation suite for Windows - built for power users.
> ![Blake Automator](icon.png)
>
> ---
> ## Features (v4.0.0 Update)
> - Auto Clicker - Repeated clicking or hold-button mode with fixed intervals or advanced anti-ban randomization.
> - - Keyboard Auto - Auto-press any keystroke dynamically at any randomized or fixed speed.
>   - - Macro Recorder - Record sequences of inputs, adjust delays, and loop infinitely.
>     - - Advanced Hotkeys - Assign any key as your global toggle hotkey with auto-cancellations.
>       - - Anti-Ban Humanizer - Integrated +/-15% randomization modes to bypass tracking.
>         - - Premium UI Themes - Switch between a resizable layout and color accents.
>           - - Profile Management - Create endless profiles to swap between configurations.
>             - - Auto-Updater - Checks for and applies new versions on launch seamlessly.
>              
>               - ---
>               ## How to Use
>               1. Download BlakeAutomator.exe from the latest release: https://github.com/sheluvsblakefrfr/blake-automator/releases/latest
>               2. 2. Run it - no installation required!
>                  3. 3. Pick Auto Clicker, Keyboard Auto, or Macro Recorder.
>                     4. 4. Set your speeds and humanizer settings.
>                        5. 5. Record your ON/OFF toggle hotkey (click the [X] to clear it).
>                           6. 6. Press your hotkey anywhere to override your system!
>
